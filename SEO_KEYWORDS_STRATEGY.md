## 7. SEO Technical Requirements

### On-Page Elements

| Element | Requirement |
|---------|-------------|
| Title Tags | Include primary keyword, keep under 60 chars |
| Meta Descriptions | Include keyword + CTA, keep under 155 chars |
| H1 | One per page, include primary keyword |
| H2-H3 | Include secondary keywords naturally |
| Image Alt Text | Descriptive, include keywords where relevant |
| Internal Links | Link to comparison page from all reviews |

### Content Guidelines

- Minimum 800 words for blog posts
- Include at least one FAQ section per article
- Add schema markup (FAQ, Product, Review)
- Update content quarterly with fresh data
